Web 2.0

The "read-write" web — participatory, social, and dynamic, where users became creators and the web became a platform for collaboration.

Definition

Web 2.0 refers to the second generation of the World Wide Web, emerging around 2004 and continuing to the present day. It represents a paradigm shift from the static, read-only Web 1.0 to a dynamic, participatory web where users actively create, share, collaborate, and interact with content. The term was popularised by Tim O'Reilly and Dale Dougherty during a 2004 conference.

Web 2.0 is not a technical upgrade to the web's standards, but rather a change in how the web is used and designed — moving from websites as static documents to the web as a platform for applications and user participation.

Key Characteristics

  • User-Generated Content (UGC) — Users create and publish their own content (blogs, videos, reviews, social posts).
  • Rich User Interfaces — AJAX, JavaScript frameworks, and CSS enable interactive, app-like web experiences.
  • Social Networking — Platforms like Facebook, Twitter, and LinkedIn enable global social connections.
  • Collaboration & Wikis — Wikipedia-style collaborative editing of shared documents.
  • Tagging & Folksonomies — User-defined tags replace rigid top-down categorisation.
  • APIs & Mashups — Open APIs allow third parties to build on and combine web services.
  • Mobile Compatibility — Responsive design serves content across devices.
Era
~2004 – present
Model
Read-Write (User ↔ Platform)
Key Tech
AJAX, REST APIs, CSS3, HTML5
Platforms
Facebook, YouTube, Wikipedia

Enabling Technologies

  • AJAX (Asynchronous JavaScript and XML) — Enables pages to update dynamically without full page reloads (used in Google Maps, Gmail).
  • REST APIs — Allow different web services to communicate and share data.
  • HTML5 & CSS3 — Richer semantic markup and advanced visual styling.
  • JavaScript Frameworks — jQuery, React, Angular, and Vue.js enable complex client-side applications.
  • Cloud Computing — AWS, Azure, and GCP enable massive scalable platforms.
"Web 2.0 is the business revolution in the computer industry caused by the move to the Internet as platform, and an attempt to understand the rules for success on that new platform." — Tim O'Reilly

Criticisms of Web 2.0

Web 2.0 also introduced significant concerns: data centralisation (a few corporations controlling vast amounts of personal data), surveillance capitalism (monetising user data through targeted advertising), misinformation spread (viral sharing of false content), platform monopolies, and privacy erosion. These challenges directly motivated the vision of Web 3.0.
