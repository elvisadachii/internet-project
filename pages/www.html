World Wide Web

The global information space where documents and resources are interlinked via hyperlinks and accessed through the Internet.

Definition

The World Wide Web (WWW), commonly known simply as "the Web," is an information system that enables the sharing of content across the Internet through a system of interlinked hypertext documents and multimedia resources. It was invented by British scientist Tim Berners-Lee in 1989 while working at CERN, the European Organisation for Nuclear Research in Switzerland.

The Web operates on top of the Internet infrastructure and is distinct from the Internet itself. While the Internet is the global network of connected computers, the Web is one of the services that runs on it — a vast collection of documents (web pages) stored on web servers and accessible via web browsers using the HTTP protocol.

History & Origin

Tim Berners-Lee submitted his proposal for the Web in March 1989. By 1991, the first website went live at CERN. The Web's key innovation was combining three existing technologies:

  • HTML — the markup language for creating web pages
  • URL — a unique address for every resource on the Web
  • HTTP — the protocol for transferring data between client and server

In 1993, CERN released the Web's source code into the public domain, enabling explosive global growth. The National Center for Supercomputing Applications (NCSA) released the Mosaic browser the same year, making the Web graphically accessible to everyday users.

How It Works

When you type a web address into your browser, the following sequence occurs:

  1. The browser sends an HTTP request to the web server hosting the page.
  2. The server locates the requested resource and sends it back as an HTTP response.
  3. The browser receives the HTML, CSS, and JavaScript files and renders the page visually.
  4. Hyperlinks within the page allow navigation to other pages or resources on any server in the world.
"The Web does not just connect machines, it connects people." — Tim Berners-Lee

Significance

The World Wide Web revolutionised communication, commerce, education, and culture. It democratised access to information, enabled e-commerce, social media, streaming, cloud computing, and fundamentally transformed every sector of society. The Web continues to evolve through Web 2.0, Web 3.0, and the emerging Web 4.0 paradigms.
