Hypertext Markup Language

The standard markup language for creating web pages — the foundational building block of every document on the World Wide Web.

Definition

HTML (Hypertext Markup Language) is the standard language used to create and structure content on the web. HTML describes the structure of a web page semantically and originally included cues for the document's appearance. Web browsers receive HTML documents from a server and render them into visible or audible web pages.

HTML uses a system of elements represented by tags (e.g., <p> for paragraphs, <h1> for headings) to annotate text. Browsers interpret these tags to display content correctly. HTML works in conjunction with CSS (for styling) and JavaScript (for interactivity) to form the complete web technology stack.

Basic HTML Structure

Every HTML document follows a standard structure:

  • <!DOCTYPE html> — Declares the document as HTML5.
  • <html> — The root element wrapping the entire document.
  • <head> — Contains metadata: page title, charset, CSS links, scripts.
  • <body> — Contains all visible page content.

Key HTML Concepts

  • Elements — The building blocks of HTML, defined by a start tag, content, and end tag: <p>text</p>.
  • Attributes — Provide additional information about elements: <img src="photo.jpg" alt="Description">.
  • Hyperlinks — Created with the <a href="..."> element — the "HT" in HTML, enabling the web's interlinked nature.
  • Semantic HTML — Using meaningful tags like <header>, <nav>, <article>, <footer> to convey document structure to browsers and assistive technologies.
  • Forms — <form> elements allow user input collection for logins, search, and data submission.
  • Media — <img>, <video>, <audio> elements embed rich media.
HTML is not a programming language — it is a markup language. It defines structure and meaning, not instructions or logic. JavaScript handles programming logic; CSS handles visual presentation.

HTML Versions

HTML has evolved from HTML 1.0 (1991) through several major versions. HTML 4.01 (1999) and XHTML (2000) formalised standards. HTML5 (2014) was a landmark update, introducing native audio, video, canvas, semantic elements, form improvements, and APIs for geolocation, local storage, and offline capabilities — transforming HTML from a document format into a full application platform.
