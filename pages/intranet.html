Intranet

A private, internal network using Internet technologies, accessible only to an organisation's authorised members.

Definition

An intranet is a private computer network that uses Internet protocols (TCP/IP, HTTP) and technologies to securely share information and resources within an organisation. Unlike the public Internet, an intranet is accessible only to employees, members, or other authorised users of the organisation.

Essentially, an intranet is a private mini-Internet for a single organisation. It may consist of multiple local area networks (LANs) and can use leased lines to span multiple physical locations. A firewall protects the intranet from unauthorised external access.

Key Features

  • Private Access — Only accessible by authenticated organisational members.
  • Internal Communication — Email systems, instant messaging, and announcements.
  • Document Management — Shared file repositories, wikis, and knowledge bases.
  • Collaboration Tools — Project management, shared calendars, video conferencing.
  • HR Portals — Payroll, leave management, onboarding resources.
  • Application Access — Internal business applications and databases.
Protocols Used
TCP/IP, HTTP, HTTPS, SMTP
Access
Authorised users only
Security
Firewall + VPN + Authentication
Scope
Internal organisation only

Benefits of an Intranet

  • Improved Communication — Centralises internal announcements and news.
  • Increased Productivity — Easy access to tools, forms, and documents.
  • Cost Reduction — Reduces paper-based processes and physical bulletin boards.
  • Knowledge Management — Creates a searchable repository of institutional knowledge.
  • Security — Keeps sensitive information away from the public Internet.
The term "intranet" first appeared in 1992, as organisations began adopting web technologies for internal use, recognising that the same tools that made the public Web powerful could revolutionise internal operations.

Intranet vs Internet vs Extranet

The Internet is the global public network open to everyone. An intranet is a private internal network. An extranet extends the intranet to selected external partners (suppliers, clients) — it sits between the two, providing controlled external access to specific intranet resources.
