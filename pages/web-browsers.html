Web Browsers

Software applications that retrieve, interpret, and display web content — the primary interface between users and the World Wide Web.

Definition

A web browser is a software application designed to access information on the World Wide Web. It acts as a client that sends HTTP requests to web servers, receives responses containing HTML, CSS, JavaScript, and other resources, and renders them as visual web pages on the user's screen.

Modern browsers are sophisticated platforms capable of running complex web applications, managing security certificates, storing cookies, and executing JavaScript code — effectively making them operating-system-like environments for web-based software.

How a Browser Works

When you enter a URL, the browser follows these steps:

  1. DNS Lookup — Resolves the domain name to an IP address.
  2. TCP Connection — Establishes a connection with the web server.
  3. HTTP Request — Sends a GET request for the page resource.
  4. Response Processing — Receives HTML, CSS, and JavaScript files.
  5. Rendering — Builds the Document Object Model (DOM) and paints pixels on screen.

Key Browser Components

  • User Interface — Address bar, navigation buttons, bookmarks, tabs.
  • Browser Engine — Marshals actions between the UI and rendering engine.
  • Rendering Engine — Parses HTML and CSS, and lays out content (e.g., Blink in Chrome, Gecko in Firefox).
  • JavaScript Engine — Executes JavaScript code (e.g., V8 in Chrome, SpiderMonkey in Firefox).
  • Networking — Handles HTTP/HTTPS requests, caching, and cookies.
  • Data Storage — Manages cookies, localStorage, sessionStorage, and IndexedDB.
The "Browser Wars" of the 1990s between Netscape Navigator and Internet Explorer shaped the standards-based web we use today.

Popular Browsers Today

Google Chrome dominates with its fast V8 JavaScript engine and integration with Google services. Mozilla Firefox prioritises privacy and open standards. Apple Safari is optimised for macOS and iOS. Microsoft Edge (rebuilt on Chromium) replaced Internet Explorer in 2020. Opera and Brave offer additional privacy and feature sets.
