Web Servers

Hardware and software systems that store web content and respond to HTTP requests from clients around the world.

Definition

A web server is a computer system — both the physical hardware and the software running on it — that stores, processes, and delivers web content to clients over the Internet or an intranet. When a web browser sends an HTTP request for a web page, the web server receives that request, locates the appropriate files, and returns them in an HTTP response.

The term "web server" can refer to both the hardware (a powerful computer in a data centre) and the software (the programme that listens for and handles HTTP requests, such as Apache or Nginx).

How a Web Server Works

  1. A browser sends an HTTP/HTTPS request to the server's IP address on port 80 (HTTP) or 443 (HTTPS).
  2. The web server software receives and parses the request.
  3. For static content (HTML, images, CSS), the server reads files from disk and returns them directly.
  4. For dynamic content, the server passes the request to an application server (PHP, Python, Node.js, etc.) which generates content on the fly.
  5. The response is sent back to the browser with a status code (200 OK, 404 Not Found, etc.).

Static vs Dynamic Servers

A static web server stores fixed files and serves them as-is. A dynamic web server consists of a web server plus extra software (typically an application server and a database) that generates customised content per request. For example, when you log into social media, a dynamic server retrieves your specific data from a database to compose a personalised page.

Popular Web Server Software

  • Apache HTTP Server — Open-source, highly configurable, used since 1995.
  • Nginx — High-performance, event-driven, excels at handling many concurrent connections.
  • Microsoft IIS — Windows-based, integrates with the .NET ecosystem.
  • LiteSpeed — Commercial server known for speed and WordPress compatibility.
  • Caddy — Modern server with automatic HTTPS via Let's Encrypt.
The world's first web server was Tim Berners-Lee's NeXT computer at CERN, bearing the label: "This machine is a server. DO NOT POWER IT DOWN!!"
