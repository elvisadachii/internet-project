File Transfer Protocol

A standard network protocol used to transfer files between a client and server over a TCP/IP network — one of the oldest Internet protocols still in use.

Definition

File Transfer Protocol (FTP) is a standard communication protocol used to transfer computer files between a client and a server on a computer network. FTP is built on a client–server model architecture using separate control and data connections between the client and the server.

Developed in the early 1970s — predating the World Wide Web — FTP remains in use today for web server file management, software distribution, and large file transfers, though it has largely been superseded by more secure alternatives like SFTP and HTTPS.

How FTP Works

FTP uses two separate TCP connections:

  1. Control Connection (Port 21) — Used for sending commands (e.g., login, list directories, change directory) and receiving server responses. This connection remains open for the entire session.
  2. Data Connection (Port 20 or dynamic) — Used for the actual transfer of file data. A new data connection is created for each file transfer or directory listing.

FTP operates in two modes: Active Mode (server initiates data connection back to the client) and Passive Mode (client initiates both connections — preferred for firewall compatibility).

FTP Variants

  • FTP — Original, unencrypted. Credentials and data sent in plain text. Not recommended over public networks.
  • FTPS (FTP Secure) — Adds SSL/TLS encryption to FTP. Two variants: Explicit FTPS (starts unencrypted, upgrades) and Implicit FTPS (always encrypted).
  • SFTP (SSH File Transfer Protocol) — Not technically FTP, but transfers files over SSH. Fully encrypted and widely preferred for security.
  • TFTP (Trivial File Transfer Protocol) — Simplified FTP using UDP, used for bootstrapping and firmware updates on embedded devices.
FTP predates the Internet as we know it — the original RFC 114 specification was published in 1971, and FTP remains one of the oldest Internet protocols still actively used today.

Common FTP Clients

Popular FTP client applications include FileZilla (open-source, cross-platform), WinSCP (Windows, supports SFTP), Cyberduck (Mac/Windows), and command-line tools like the built-in ftp command in Linux/macOS. Many web hosting control panels (cPanel, Plesk) also include integrated FTP file managers.
